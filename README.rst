Reflog Dashboard
================

A super simple dashboard for seeing the most talked about GitHub repos.


Powered by
----------

* `Flask <http://flask.pocoo.org/>`_
* `Redis <http://redis.io>`_
* `ep.io <http://ep.io>`_
* `skeleton <https://github.com/dhgamache/Skeleton>`_
